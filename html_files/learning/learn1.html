    Abstract

    Comparing to classic control theory, modern control theory has a steeper view since the subject of study focuses on state variables rather than inputs and outputs. The state variables, corresponding to the degrees of freedom of a system, provides the information of the system's inner structure. Based on matrix theory, especially matrix factorization techniques, the linear time-invariant systems can be better analyzed and controlled.

    The mathematical framework of this subject is similar to that in Classic Control Theory, containing calculus, integral transformation, and linear algebra in a large content. In the last chapter, optimal control, convex optimization is also required.

    Keywords: Modern Control Theory, State Space, Lyapunov, Controllability, Observability

    线性控制系统的能控性和能观性

    能控性： ∃ 一个分段连续的输入 u ( t ) ，能在有限时间区间 [ t 0 , t f ] 内，使系统由某初始状态 x ( t 0 ) 转移至 ∀ 指定终态 x ( t f ) ，则状态 x ( t 0 ) 能控；
    系统能控充要条件：
     1.特征值互异： T − 1 B 各行不全为0；
     特征值相同部分：约旦块最后一行相应 T − 1 B 行不全为0；
     2. W u x ( s ) 无零极点重合现象；
     3.能控性矩阵 M = [ B   A B   …   A n − 1 B ] 秩为 n ；

    能观性：对 ∀ 给定输入 u ，在有限观测时间，使根据 [ t 0 , t f ] 的 y ( t ) 可唯一地确定 x ( t 0 ) ，则状态 x ( t 0 ) 可观测；
    系统能观的充要条件：
     1. A ∼ Λ ： C 中无全为0的列；
      A ∼ J ：每个约旦块开头的一列元素不为0；
     2.能观性矩阵 N = C C A ⋮ C A n − 1 秩为 n

    稳定性与李雅普诺夫方法

    李雅普诺夫稳定性：对 ∀ ε > 0 ,   ∃ δ ( ε , t 0 ) > 0 ,   x 0 ∈ U ( x e ; δ ) 时， Φ ( t ; x 0 , t 0 ) ∈ U ( x e ; ε ) , t ≥ t 0 ，则系统李雅普诺夫稳定；
    李雅普诺夫一致稳定： δ 不依赖于 t 0 ；
    渐近稳定：系统李雅普诺夫稳定，且 lim t → ∞ x ( t ) = 0 ；
    输出稳定：系统由有界输入引起的输出有界；
